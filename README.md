# 蜘蛛纸牌（Spider Solitaire）

一个**纯前端、无需服务器**的蜘蛛纸牌小游戏：直接打开 `index.html` 就能玩（离线可用）。

## 功能

- 经典 10 列布局 + 库存 5 次发牌
- 难度：1 / 2 / 4 花色
- 拖拽移动：仅允许移动「同花色连续降序」的牌段
- 自动翻开暗牌
- 自动收走完整序列（同花色 K→A）
- 撤销（Undo）

## 运行

### 方式 1：直接打开

双击打开：`index.html`

### 方式 2：用任意静态服务器（可选）

例如：

```bash
python3 -m http.server 5173
```

然后打开：<http://localhost:5173>

## 操作

- 拖动一段可移动的牌到目标列
- 点击“发牌”或点击左上库存发牌
- `Ctrl/Cmd + Z` 撤销

见页面右上角“规则”。
